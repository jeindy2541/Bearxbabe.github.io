      คิวยังว่าง
      วางลิงก์ YouTube ด้านบนแล้วกด “ขอเพลง”
      ยังไม่มีเพลงที่บันทึก
      กด GRAB ตอนเพลงเล่นเพื่อเก็บไว้

      ตั้งค่า Firebase ก่อนใช้งาน 🔧

      ต้องใส่ค่า Firebase ของคุณ (ฟรี) เพื่อให้ห้องออนไลน์เชื่อมต่อกันได้

      1. ไปที่ console.firebase.google.com → สร้างโปรเจกต์ใหม่ (ฟรี)
      2. เมนู Build → Realtime Database → Create Database → เลือก Start in test mode
      3. กดไอคอน </> (Add web app) → คัดลอกค่า firebaseConfig
      4. เปิดไฟล์นี้ด้วย text editor → วางค่าทั้งหมดในช่อง firebaseConfig ด้านบนสุดของ <script> (รวม databaseURL)
      5. เซฟไฟล์ แล้วอัปโหลดขึ้นโฮสต์ฟรี (Netlify Drop / GitHub Pages / Firebase Hosting) แล้วแชร์ลิงก์ให้ทีม

      ทำครั้งเดียวจบ หลังจากนั้นทุกคนแค่เปิดลิงก์เดียวกันก็เข้าห้องได้เลย

      เข้าห้องเปิดเพลง 🎧

      ตั้งชื่อที่จะให้คนในห้องเห็น แล้วกดเข้าห้องเพื่อเริ่มฟังพร้อมกัน
